TokiDesign Utility Library — Full Demo

Class-based utilities for layout, shapes, transforms, filters, animations, and more

What is this?

Use simple classes like g_tc_1fr_2fr, anim_td_fade_in_600ms_ease_1, tf_r_45deg, flt_blur_4px to style elements without writing CSS.

How to use

Add classes to your HTML elements. The library parses tokens and applies inline styles or injected CSS rules.
